BOM y DOM
BOM: Browser Object model

El BOM está compuesto por varios objetos relacionados entre sí.
objeto global = window
window : document/frames/history/location/navigator/screen
window.document: anchors/forms/images/links/location

dom(?):
Document Object Model
ejemplo de nodo:
document.createElement
Arbol:
document -> html -> body, title
title -> meta/title/"text"
body -> h1/"text"/ul/a/href/li

DomScripting:
DOM Scripting" means using JavaScript to manipulate the DOM (page elements and contents) inside a web page, either during page load, or in response to user events, like clicking a button or selecting text.

*obtener el div por id:
document.getElementById('')
*por clase:
document.getElementByClassName('')
*por etiqueta:
document.getElementByTagName
*por selector(devuelve el primero):
document.querySelector('')
devolver toda la lista que coincidan = document.querySelectorAll()

La propiedad textContent te permite acceder al contenido del nodo donde estemos trabajando(solo texto). representa el contenido del texto.

para representar el contenido -> innerText. otra forma de acceder al elemento del nodo

**Usar innerHTML no es una forma segura, ademas de que no crea contenido.

DOMScripting:

/*DOM scripting */
/*
Crear elementos
*/

const nuevoDiv= document.createElement('div');
nuevoDiv.classList.add('item');

const h2 = document.createElement('h2');
h2.innerText = 'Elemento 2';
nuevoDiv.appendChild(h2);

const p2 = document.createElement('p');
p2.textContent = 'Descripcion del elemento 2';
nuevoDiv.appendChild(p2);

const contenedor = document.getElementById('contenedor');
contenedor.appendChild(nuevoDiv);

Eventos:
Son la ejecuciòn de funciones como respuesta a una acciòn y se envian para notificar el codigo de cosas interesantes que han ocurrido. Cada evento està representado por un objeto que se basa en la interfaz Event y puede tener campos y/o funciones personalizadas adicionales para obtener mas informacion acerca de los sucedido.
